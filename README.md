# Varejo_Max - análise de dados
Script em Python de análise de dados de uma empresa fictícia - Varejo Max

Script obtém os dados de 4 arquivos .CSV e retorna 9 arquivos em .xlsx com ranking de vendedores, produtos, região, além de vendas por mês e uma tabela unificada. Retorna também dois gráficos: vendas por mês e ranking de vendedores pelo faturamento.
